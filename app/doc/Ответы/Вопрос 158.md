## <u>Кратко</u>

Для функционально-ориентированного программирования в Python часто используют методы и функции: `map()`, `filter()`,
`reduce()`, `lambda`-функции и `functools.partial`.

## <u>Развернуто</u>

Функционально-ориентированное программирование (ФОП) в Python основано на использовании функций как объектов первого
класса и на применении чистых функций, неизменяемости данных и композиции функций. В Python есть несколько встроенных
инструментов для реализации принципов ФОП.

1. **`map(function, iterable, ...)`**
    - Применяет функцию к каждому элементу последовательности и возвращает итератор с результатами.
    - Позволяет лаконично преобразовывать коллекции данных без использования циклов.
    ```python
    numbers = [1, 2, 3, 4]
    squared = list(map(lambda x: x ** 2, numbers))
    print(squared)  # Вывод: [1, 4, 9, 16]
    ```

2. **`filter(function, iterable)`**
    - Отбирает из последовательности элементы, для которых функция возвращает `True`.
    - Используется для фильтрации данных.
    ```python
    numbers = [1, 2, 3, 4, 5]
    even = list(filter(lambda x: x % 2 == 0, numbers))
    print(even)  # Вывод: [2, 4]
    ```

3. **`reduce(function, iterable)`** (из модуля `functools`)
    - Последовательно применяет функцию к элементам последовательности, сводя их к единственному значению.
    - Полезна для свёртки (агрегации) данных.
    ```python
    from functools import reduce
    numbers = [1, 2, 3, 4]
    product = reduce(lambda x, y: x * y, numbers)
    print(product)  # Вывод: 24
    ```

4. **`lambda`-функции**
    - Анонимные однострочные функции, удобны для коротких операций в `map()`, `filter()` и других местах.
    - Позволяют писать компактный и функциональный код.

5. **`functools.partial`**
    - Позволяет фиксировать некоторые аргументы функции, создавая новую функцию с меньшим числом параметров.
    - Упрощает композицию функций.
    ```python
    from functools import partial
    def power(base, exponent):
        return base ** exponent

    square = partial(power, exponent=2)
    print(square(5))  # Вывод: 25
    ```

6. **Преимущества и недостатки**
    - **Преимущества:**
        - Код становится более выразительным и компактным.
        - Способствует написанию чистых функций, облегчая тестирование и поддержку.
    - **Недостатки:**
        - Иногда функциональный код может быть менее понятен новичкам.
        - Использование встроенных функций с лямбдами иногда приводит к усложнению отладки.

7. **Контекст использования**
    - Функционально-ориентированные методы широко используются при обработке данных, построении конвейеров обработки, в
      библиотеке pandas, numpy, а также при создании сложных трансформаций.

Таким образом, в Python для реализации функционально-ориентированного программирования применяются `map()`, `filter()`,
`reduce()`, `lambda`-функции и другие инструменты, позволяющие писать лаконичный и выразительный код.
