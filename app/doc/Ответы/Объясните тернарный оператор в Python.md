## <u>Кратко</u>

Тернарный оператор в Python — это краткий способ записи условного выражения, позволяющий выбрать одно из двух значений в
зависимости от условия.

## <u>Развернуто</u>

Тернарный оператор используется для присвоения значения переменной или возврата результата в зависимости от логического
условия, при этом запись занимает всего одну строку. Формат записи следующий:

```python
value_if_true if condition else value_if_false
```

1. **Описание работы:**
    - Сначала вычисляется `condition`.
    - Если условие истинно (`True`), результатом становится `value_if_true`.
    - Если условие ложно (`False`), результатом становится `value_if_false`.

2. **Пример использования:**
    ```python
    age = 20
    status = "Взрослый" if age >= 18 else "Ребенок"
    print(status)  # Вывод: Взрослый
    ```

3. **Преимущества и недостатки:**
    - **Преимущества:**
        - Краткая и компактная запись простых условий.
        - Повышает читаемость при использовании в простых случаях.
    - **Недостатки:**
        - Потенциально ухудшает читаемость при сложных или вложенных условиях.
        - Для сложной логики лучше использовать традиционный `if-else`.

4. **Контекст использования:**
    - Тернарный оператор удобен для быстрой и простой логики выбора значений, особенно при инициализации переменных.

5. **Еще пример:**
    ```python
    score = 85
    result = "Успешно" if score >= 60 else "Неудачно"
    print(result)  # Вывод: Успешно
    ```

Таким образом, тернарный оператор — это лаконичный инструмент для работы с условными выражениями в Python, позволяющий
записывать их в одну строку.
