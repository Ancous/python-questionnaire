но, (_) можно использовать подчеркивание в качестве переменной в цикле. Примеры ниже:

Пример 1

```python

for _ in range(5):
    print(_)
```

```commandline
0
1
2
3
4
```

Пример 2

```python

languages = ["Python", "JS", "PHP", "Java"]
for _ in languages:
    print(_)
```

```commandline
Python
JS
PHP
Java
```

Пример 3

```python

_ = 5
while _ < 10:
    print(_, end = ' ') 
    _ += 1
```

```commandline
5 6 7 8 9 
```
